# NariRaksha Women Safety App

NariRaksha is a women safety application that helps users stay safe by detecting unsafe locations and sending emergency alerts.

## Features

- GPS Location Tracking
- Red / Yellow / Green Safety Zones
- SOS Emergency Alert
- Sends Live Location to Emergency Contacts
- Police Station Notification System
- Google Maps Integration

## Technologies Used

- HTML
- CSS
- JavaScript
- Google Maps API
- Local Storage Database

## How It Works

1. User logs into the application
2. The app detects the user's GPS location
3. The system checks the safety zone
4. Alerts are shown for unsafe areas
5. SOS button sends location to emergency contacts
